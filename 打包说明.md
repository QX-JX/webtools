# 全能站长工具 - 打包说明

## 快速开始

1. **一键打包**
   ```bash
   build-package.bat
   ```

2. **生成的安装程序**
   - 位置: `electron\build_release_v13\全能站长工具-Setup-1.0.1.exe`
   - 图标: 使用 `public/app.ico`
   - 支持静默安装

## 安装程序使用方法

### 普通安装
```bash
# 双击运行，图形界面安装
全能站长工具-Setup-1.0.1.exe
```

### 静默安装
```bash
# 静默安装到默认位置，自动创建桌面快捷方式
"全能站长工具-Setup-1.0.1.exe" /S
```

### 静默安装到指定目录
```bash
# 静默安装到自定义目录
"全能站长工具-Setup-1.0.1.exe" /S /D=C:\MyApps\全能站长工具
```

### 静默卸载
```bash
# 静默卸载，自动清理快捷方式
"C:\Program Files\全能站长工具\Uninstall 全能站长工具.exe" /S
```

## 静默安装特性

✅ **自动创建桌面快捷方式**
✅ **自动创建开始菜单快捷方式**  
✅ **无需用户交互**
✅ **支持批量部署**
✅ **自动注册到系统程序列表**
✅ **卸载时自动清理所有文件和注册表**

## 技术细节

- **打包工具**: electron-builder + NSIS
- **图标文件**: public/app.ico
- **支持架构**: x64, ia32
- **目标平台**: Windows 7+
- **安装权限**: 需要管理员权限

## 文件结构

```
electron/
```

## 注意事项

1. **本地网页开发（http://localhost:5888）**：需同时启动后端 API（默认 `http://localhost:3888`），否则浏览器里访问 `/api/*`（含登录代理）会失败。请使用项目自带的 `npm run dev`（会拉起 Vite + 后端编译监听），或单独运行 `npm run dev:server` 后再启动前端。
2. 确保所有依赖已安装 (`npm install`)
3. 构建前会自动清理旧文件
4. 静默安装需要管理员权限
5. 图标文件必须是 .ico 格式
6. 支持中文路径和文件名
